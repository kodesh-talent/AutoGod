# AutoGod
AutoGod-自动化助手-安卓自动化测试平台，官方发布主页



## AutoGod-是什么？

AutoGod（自动化助手）是一款面向开发者和自动化测试人员的新一代Android自动化测试框架。它以安全性为核心设计理念，集成了强大的AI识别能力、极致的性能表现以及全面的防破解保护机制，帮助用户高效构建、运行和保护自动化脚本。无论你是个人开发者还是企业团队，AutoGod都能为你提供从脚本编写、调试到打包部署的一站式解决方案。



## AutoGod-能干什么？

App自动化测试：快速编写回归测试、UI自动化测试脚本。

游戏辅助自动化：利用OCR + YOLO识别实现游戏内自动操作。

企业RPA流程：构建跨应用的自动化业务流程。

脚本商业化保护：通过多层安全机制保护你的脚本知识产权。



## AutoGod-的优势？

支持Shizuku、Root、无障碍、HID等多策略；

内置Yolo全系列+主流OCR；

Material Design3精美UI开发；

JS代码经简化、混淆、VMP转码三重保护，APK高度防逆向；

专为开发者优化，兼顾易用性与安全性。



拓展性强：支持加载APK、DEX、JAR、SO文件，并提供Plugin插件开发机制，灵活扩展功能。

兼容性强：覆盖Android 5+至Android 16全版本系统，完美适配雷电模拟器、云手机等平台。

完善的文档体系：提供API英文/中文文档、UI英文/中文文档，上手无门槛。

专业开发工具：内置IDEA开发插件，提供开发者后台管理系统。
